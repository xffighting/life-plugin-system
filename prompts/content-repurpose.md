# 内容复用提示词 · 一鱼多吃 / Content Repurpose

> 用途：一条核心素材（一鱼）→ 自动改写成多个平台各自调性的版本（多吃）。
> 适合：做自媒体、知识分享、个人品牌的人。一次创作，多平台分发。

---

## 提示词（复制这一整段）

```
你是我的多平台内容改写助手。我给你一条核心素材，请把它改写成四个平台各自调性的版本。
注意：不是一稿四发，每个平台的语气、长度、结构都要不同。

# 我的核心素材
- 核心观点（一句话）：【...】
- 给读者的钩子（他为什么要看）：【...】
- 支撑要点（2-3 条）：【...】

# 请输出四个版本：
1. 小红书：第一人称痛点开头 + 适度 emoji + 分点 + 互动式结尾。短、有共鸣。
2. 公众号：完整方法论长文。标题有吸引力但不标题党，结构清晰、有小标题。
3. 视频号口播脚本：30–60 秒。前 3 秒一个强钩子，结尾引导关注。
4. 抖音口播脚本：15–30 秒。开头钩子更猛，配分镜/字幕提示。

# 要求
- 四个版本的开头必须各不相同。
- 不要出现任何真实客户、真实价格、财务数字、私人信息。
- 只产出文案/脚本，不替我拍摄或发布。
```

---

## 质量自检（发布前）

- [ ] 四版开头各不相同（不是复制粘贴）。
- [ ] 每版都有独立的钩子。
- [ ] 无真实客户/价格/隐私（可用本仓库 `automations/deid_gate.py --gate 草稿.md` 扫一遍）。
- [ ] 视频版只是脚本，发布动作由你人工完成。

> 示例素材（虚构）：观点=「普通人也能给人生装个外挂」；钩子=「我把自己的人生操作系统开源了」。
