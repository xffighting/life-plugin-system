# 安全与隐私

这个项目的核心前提是：**真实的人生数据永远留在你本地，公开的只有模板与方法。**

## 给使用者：保护好你自己的数据

1. **不要**把真实客户、报价、合同、收入、健康、家庭、聊天记录、邮件原文放进任何公开仓库。
2. 把你的真实 Vault 放在本仓库**之外**；本仓库只放通用模板和虚构示例。
3. 发布/分享任何内容前，先跑脱敏检查：
   ```
   python3 automations/deid_gate.py --gate 你的草稿.md
   ```
   退出码为 0 才算干净。
4. 想把含真实姓名/公司的草稿脱敏，用：
   ```
   python3 automations/deid_gate.py --sanitize 草稿.md --out 草稿.public.md --names "你的名字,你的公司"
   ```

## 隐私分级（建议）

给每个笔记的 frontmatter 标一个 `privacy` 等级：

| 等级 | 含义 | 能否公开 |
|---|---|---|
| `public` | 可公开 | 是 |
| `internal` | 内部/脱敏后可分享 | 视情况 |
| `private` | 个人私域、家庭、健康 | 否 |
| `sensitive` | 客户/财务/合同 | 否，仅本地 |

`private` 与 `sensitive` 的内容**不要**进入任何公开仓库。

## 脱敏脚本能做什么、不能做什么

`automations/deid_gate.py` 能自动抓：邮箱、电话、金额、附件直链、网盘链、账户凭证。

它**抓不到**（需你人工复核）：未列入黑名单的真实姓名、即时通讯号、证件号、银行卡号、
物理地址、git commit message、文件名里的敏感信息、图片里的文字。

## 给贡献者：报告安全问题

如果你发现示例数据里混入了疑似真实信息，或脚本有泄漏风险，请开一个 Issue
（敏感情况可注明"请私下联系"），我们会尽快处理。
