# 人生外挂系统 · Life Plugin System

> **Life Plugin System is an open-source life operating kit for AI-assisted personal workflows.**
> 人生外挂系统是一套开源的、AI 辅助的个人工作流与人生系统模板。把散落的目标、项目、复盘、知识、关系和 AI Agent 协作，整理成一个能持续生长的"外部大脑"。

<p align="center">用 Obsidian 打开 · 用 AI Agent 驱动 · 用复盘校准 · MIT 许可，随便用</p>

---

## 为什么会有这个项目

我是一个做外贸的普通人。每天有处理不完的询价、记不住的人、想做却总忘的目标、读了就忘的书。

我试过很多工具，最后发现问题不在工具，而在于：**我的人生没有一个"系统"——一个能把事实、判断、行动、关系连成一张网，还能让 AI 帮我一起维护的系统。**

于是我用 Obsidian 搭了一套：文件夹负责存放，图谱负责连接，目标负责牵引，项目负责推进，复盘负责校准，AI 负责协同。

我把其中**不涉及隐私的那一层**抽出来开源，希望它也能成为你的"人生外挂"。

---

## 它解决什么问题

- 目标立了就忘 → 每个目标都有指标和证据，并标注它靠**哪种杠杆、能否复利**。
- 项目散乱 → 每个项目必须服务一个目标，否则归档。
- 知识读了就忘 → 知识沉淀 + 双链连接，越用越值钱。
- AI 用得随意 → 把提示词和 Agent 技能固化成可复用资产。

---

## 适合谁 / 不适合谁

**适合**：想给人生建一套"操作系统"的人；Obsidian 用户；想让 AI Agent 真正帮上忙的人；做内容/外贸/投资需要长期积累的人。

**不适合**：只想要一个待办清单的人；不愿花 1 小时建立结构的人；期待"装上就改变人生"的人——**这是外挂，不是魔法**。

## 5 分钟快速开始

1. 安装 [Obsidian](https://obsidian.md)（免费）。
2. 下载本仓库，把 `obsidian/sample-vault/` 作为一个 Vault 打开。
3. 打开 `obsidian/sample-vault/START_HERE.md`，跟着走。
4. 从 `templates/goal.md` 拷一份，建你的第一个目标。
5. 把 `prompts/daily-review.md` 丢给你的 AI 助手，开始第一次复盘。

## 目录结构（当前 MVP）

```
templates/     空白模板（目标卡片，含杠杆/复利字段）
prompts/       即用提示词（每日复盘、内容复用）
automations/   低风险脚本（脱敏检查 deid_gate.py）
obsidian/      虚构示例 Vault（5 分钟跑起来）
governance/    贡献规范、安全规范
ROADMAP.md     路线图（build-in-public）
```

> 这是早期 MVP，更多模板/技能/文档正在持续加入，见 `ROADMAP.md`。

## 隐私安全（请认真读）

本项目的设计前提是：**真实的人生数据永远留在你自己本地，公开的只有模板与方法。**

- 不要把真实客户、报价、合同、健康、家庭、聊天记录提交到任何公开仓库。
- 发布前先跑 `automations/deid_gate.py --gate 你的文件` 做脱敏检查。
- 详见 `governance/SECURITY.md`。
