## 先问你一个扎心的问题

大多数人玩"人生"这款游戏,用的是**最硬核的模式**:

- 没有存档——错了不能读档重来;
- 没有攻略——边玩边猜;
- 没有系统面板——不知道自己点了哪些技能、哪条线快没血了。

然后我们一边骂这游戏难,一边继续裸奔。

**这套开源工具包,就是你的系统面板 + 外挂 + AI 队友。** 不是又一个待办 App,是一整套能运行、能自我进化、还能让 AI 帮你一起打的**人生操作系统**。

下面把后台拆开给你看。看完你大概率会有个冲动:**也想给自己开一个。**

## 你将装备的七件套

### ① 一部宪法 —— 不会被情绪改写的底层规则

人生最大的 bug,是规则天天变:今天发奋,明天躺平。你不是没努力,是**没有一套不被情绪改写的底层设定**。

所以第一件事,是给人生立法。文件可以重写,宪法只能修订:

> **复利信念**:每天 1% 的清晰,胜过每月一次的重整。
> **事实优先**:先记录事实,再做判断,再决定下一步。情绪不替代事实。
> **人胜过系统**:当规则与生活冲突,先服从生活,再修订系统。系统不能让人变成它的奴隶。

修订还得走流程(写动机、同步字典、升版本号),**不接受口头修改**——直接干掉"我又拍脑袋换打法"这个最大内耗源。

### ② 八条技能树 —— 别只点一棵树

RPG 里你不会把点数全加在攻击力上。现实里我们却常常只点"事业",结果"健康""家庭"两条线悄悄红血濒死。

系统把人生切成**八条经纬**(身体/心理/家庭/事业/投资/学习/人际/表达),等于八棵必须长期点的技能树。规则很硬:**任何一条连续两个月没更新,系统直接报警。** 这是给人生装的**八维血条仪表盘**——哪条在掉血,一眼可见。

### ③ 三权 + 五节律 —— 主线机制与存档点

- **目标**是方向盘,**项目**是发动机,**复盘**是校准器。三者不能混用,混用=违宪。
- 五个存档节律:日省 / 周省 / 月省 / 季省 / 岁省。你不再靠"突然的觉醒"纠偏——系统每天每周每年都帮你对一次表。

### ④ 开挂引擎:杠杆 × 复利 —— 把纳瓦尔做成排序算法

前面让游戏跑得稳,这一件让你**开挂**。每个目标/项目都强制带四个字段:

| 字段 | 它逼你回答 |
|---|---|
| `leverage` | 靠哪种杠杆?(判断 > 媒体/代码 > 资本 > 劳力) |
| `asset_type` | 在攒什么资产?(受众/专长/系统/股权/关系) |
| `compounding` | 一次投入,是否持续产出? |
| `passive_income` | 你睡着时,它还在为你赚钱吗? |

于是系统会把"睡后还在产出的事"自动顶到排期最前面。**财富是资产,不是工资。这,就是外挂。**

### ⑤ 逆向开挂:一张不做清单

高手不找"还能加什么 buff",而是先排掉 debuff。芒格说"我只想知道我会死在哪,然后永远不去"。**删掉一件消耗,常比新增一件更值钱**——这是大多数人从没用过的开挂键。

### ⑥ 一支 AI 舰队 —— 你只做判断

系统不是你一个人在玩。它为 AI Agent 而生:`prompts/` 即拿即用,`skills/` 自带 输入/输出/步骤/质检,[入网七步](docs/architecture.md)是给 Agent 执行的状态机,[脱敏硬闸](automations/deid_gate.py)在产出对外前命中即拦。

但铁律写进了宪法:**AI 是协作者,不是裁判。** 把机械重复全外包给舰队,把自己解放到只做"判断"这一件最高杠杆的事上——**判断力,是 AI 时代唯一不贬值的资产。**

### ⑦ 一张网,不是一张清单

待办清单是线性的、会过期的、彼此孤立的——你今天勾掉的 100 件事,三个月后什么都不剩。这套系统是一张**网**:事实、目标、项目、人、知识、复盘全部互连。一件事做完不是"勾掉消失",而是**沉淀成网络里的节点,继续给未来的判断供能**。你的人生数据,第一次开始利滚利。

## 为什么它"用十年也不乱"

普通模板三个月就成垃圾场,因为没有治理。这套自带免疫系统:概念字典消灭模糊词、入网七步约束每份新内容、健康度指标定期体检(孤立节点 < 5%、目标-项目连接率 100%)。**越用越清晰,而不是越用越乱。** 想看工程内核 → [docs/architecture.md](docs/architecture.md);数据结构(双轨制)→ [docs/schemas.md](docs/schemas.md)。

隐私默认本地:五级隐私 + 白名单导出 + fail-closed。**真实人生数据永远留在你本地,公开的只有方法与模板。**

## 今晚就能开挂:先抄三步

不用一上来就建宪法。今晚就能动手:

1. **画八条技能树,量一次血条。** 八个领域各打 1–10 分,哪条最低,下周补哪条。
2. **写三条不做清单。** 把过去一年最大的三个坑,写成"我不再……"。
3. **挑一件"睡后还在产出"的事,今天就开始。** 一篇会被持续搜索的文章、一个可复用的脚本、一条长期有效的内容。

然后,正式开挂:

```
安装 Obsidian → 打开 obsidian/sample-vault/
→ 读 00_美满总图/START_HERE_从这里开始.md（六步导航）
→ 拷 templates/goal.md 建第一个目标 → 把 prompts/daily-review.md 交给你的 AI
```

## 目录结构

仓库本身:

```
docs/          架构与设计思想（先读这里）
templates/     空白书写模板（目标/项目/复盘/人物/知识 + 目录导览模板）
prompts/       即用提示词（每日复盘、内容复用）
skills/        AI Agent 技能（含入网七步协议）
automations/   低风险脚本（发布前脱敏硬闸）
obsidian/      虚构示例 Vault（完整复刻下面的架构）
governance/    贡献规范、安全与隐私
```

示例 Vault(`obsidian/sample-vault/`)**完整复刻了这套"五层 + 一脊"架构**,你打开就是一套能直接用的骨架:

```
00_美满总图    内核：宪法 + 总图 + START_HERE
10_生命经纬    脊柱：八条人生经纬（八维血条）
15_目标罗盘    L2 目标（方向盘）        20_行事务    L4 项目（发动机）
30_万象入流    L1 输入                  35_人际经纬  L1 人物与关系
40_知行成库    L3 知识资产              45_能力资产  L3 能力五形态
50_智能协同    脊柱：Agent/工具/渠道    60_数据账册  脊柱：CSV 注册表（双轨制）
70_四时复盘    L5 复盘（校准器）        80_图谱索引  脊柱：图谱治理 + 健康度
85_系统字典    脊柱：概念治理           90_模板雅集  工具：模板
91_附件物藏    工具：附件               99_旧事归档  工具：归档
```

> 每个目录里都有一份 `_导览.md`,讲清"这里收什么、不收什么"。真实的客户/邮件/数据账册等只留本地,公开版只复刻**结构**。
