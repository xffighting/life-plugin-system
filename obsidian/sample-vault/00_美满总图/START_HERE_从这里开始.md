# START HERE · 从这里开始

> 欢迎打开「人生外挂系统」示例 Vault。所有内容都是**虚构**的,用来演示这套"五层一脊"架构怎么运转。
> 你可以删掉示例,换成自己的真实内容(真实内容建议设 `privacy: private`,不要公开)。

## 第一次打开,走六步

1. 看总图:[[MOC_总图]],理解"五层 + 一脊"。
2. 看目标罗盘:`15_目标罗盘/_导览`,确认本周真正要推进的目标。
3. 看行事务:`20_行事务/_导览`,只挑 1–3 个 active 项目推进。
4. 新增内容先入网:新资料/想法先进 `30_万象入流`,再按入网七步归位。
5. 每日复盘:用 `templates/review.md` 记录事实、判断、下一步。
6. 每周体检:看 `80_图谱索引/_导览` 的健康度指标,清理孤立节点。

## 今天只做三件事

- 更新一个目标。
- 推进一个项目。
- 归位一个新输入。
