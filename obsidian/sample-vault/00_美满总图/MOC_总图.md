# MOC · 总图(五层 + 一脊)

> 这是整个 Vault 的总入口。结构 = 一个内核 + 五层数据流 + 一根贯穿的脊柱。

```
            ┌──────────────────────────────┐
            │  内核 L0 · 00_美满总图         │  宪法 + 总入口
            └──────────────────────────────┘
                          │
  L1 输入 ──▶ L2 处理 ──▶ L3 沉淀 ──▶ L4 产出 ──▶ L5 反馈
 30_万象入流  15_目标罗盘  40_知行成库  20_行事务  70_四时复盘
 35_人际经纬              45_能力资产
                          │
  ── 脊柱：10_生命经纬 · 50_智能协同 · 60_数据账册 · 80_图谱索引 · 85_系统字典 ──
                工具：90_模板雅集 · 91_附件物藏 · 99_旧事归档
```

## 十六目录角色矩阵

| 目录 | 层级 | 角色 |
|---|---|---|
| `00_美满总图` | L0 | 宪法 + 总入口 |
| `10_生命经纬` | 脊柱 | 八条人生经纬域 |
| `15_目标罗盘` | L2 | 目标(方向盘) |
| `20_行事务` | L4 | 项目(发动机) |
| `30_万象入流` | L1 | 输入流 |
| `35_人际经纬` | L1/沉淀 | 人物与关系 |
| `40_知行成库` | L3 | 知识资产 |
| `45_能力资产` | L3 | 能力五形态 |
| `50_智能协同` | 脊柱 | Agent/工具/渠道 |
| `60_数据账册` | 脊柱 | CSV 注册表 |
| `70_四时复盘` | L5 | 复盘(校准器) |
| `80_图谱索引` | 脊柱 | 图谱治理 |
| `85_系统字典` | 脊柱 | 概念治理 |
| `90_模板雅集` | 工具 | 模板 |
| `91_附件物藏` | 工具 | 附件 |
| `99_旧事归档` | 工具 | 归档 |

> 判定口诀:先看"是输入/处理/沉淀/产出/反馈",再看"指向八经纬哪一条",最后看节点类型。
